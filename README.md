# Clone do Spotify

Este é um Clone do Spotify, desenvolvimento para 
fins educacionais. Objetivo é criar uma interface semelhante á 
do Spotify, utilizando tecnologias web modernas.
 
 ## Tecnologias Utilizadas

 -HTML 5
 -CSS 3
 -Javascript
 -Tailwind CSS (versão 4.x)
 - Node.js (versão 24.x)

 ## Instalação
 1. Clone este repositório
 ```bash
 git clone https://github.com/mayarabatista/CloneSpotify.git

 2. Navegue até o diretório do projeto:
 ```bash
 cd CloneSpotify
 ```

 3. Instale as dependências:
 ```bash
 npm install
 ```
